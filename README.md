# Getting and Cleaning Data Project

## Overview
This project demonstrates data cleaning using the UCI HAR dataset.

## Files

- run_analysis.R : Script to clean and process data
- tidy_data.txt : Final tidy dataset
- CodeBook.md : Description of variables and transformations

## How to run

1. Download dataset
2. Place inside a folder named 'data'
3. Run run_analysis.R

## Output

The script generates tidy_data.txt
